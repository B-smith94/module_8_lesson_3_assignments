Building Responive Layouts with CSS Grid

Thank you for viewing. This is the header!

Content

This is where the bulk of the main content would go. It will probably be filled with paragraphs and the like. It may run on for a long time as well! It might say things like "Hello world!" or "Would you like to build a snowman?" if it is appropriate for the subject at hand. You could put almost anything here! But remember, it must relate to the subject matter of the web page, so as not to confuse the users.

In the CSS section, we have it set up so this will appear on the left, and the sidebar will appear on the right.

Sidebar

This is the sidebar. It might have supplemental information relating to the Content portion of the page.

As stated before, this will appear on the right, thanks to our CSS Grid Layout.
